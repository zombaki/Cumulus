Batch Data Import

These classes handle the NPSP Batch Data Import functionality that allows Administrators to process records in the denormalized DataImport__c object into Accounts, Contacts, Addresses, Opportunities, and Payments:

  • BDI_BatchOverride_CTRL - the page controller for Batch Edit/New/Clone override
  • BDI_BatchOverride_TEST - test class
  • BDI_BatchDataImport_API - the official global and invocable methods for BDI
  • BDI_DataImport_BATCH - the batchable class that implements the Data Import process
  • BDI_DataImport_CTRL - the page controller for the Data Importer page
  • BDI_DataImport_TEST - test class
  • BDI_DataImport_TEST2 - test class
  • BDI_DataImportAPI_TEST - test class
  • BDI_DataImportBatch_SCHED - the schedulable job to process Data Import Batches that are flagged for processing
  • BDI_DataImportBatch_SEL - selector class for the Data Import Batch object
  • BDI_DataImportBatch_TEST - test class
  • BDI_DataImportCTRL_TEST - test class
  • BDI_DataImportDeleteBTN_CTRL - page controller for the Delete Data Import records button
  • BDI_DataImportDeleteBTN_TEST - test class
  • BDI_Donations - handles the importing of Data Import donations
  • BDI_Donations_TEST - test class
  • BDI_IMatchDonations - Interface that defines the matching logic for Donations
  • BDI_MatchDonations - Our implementation of the IMatchDonations interface
  • BDI_SettingsUI_CTRL - the page controller for the visualforce component BDI_SettingsUI, which provides a UI for editing processing settings
  • BDI_SettingsUI_TEST - test class
