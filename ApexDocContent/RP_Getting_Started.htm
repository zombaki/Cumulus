Getting Started Page

Page implemented to provide users information for NPSP available resources. These resources include Youtube videos, GitHub release information, amongst other links, in order to help and provide assistance to the Salesforce community.

This new implementation include a page to be used as home pagem, which provides the following resources

  • A section to include links refer NPSP information.
  • Youtube section including videos related to NPSP functionality.
  • Github section that display the lastest release notes, version and url to download the package, from salesforce.org Cumulus repo in Github.

Artifacts implemented for this module

Visualforce Page included:

  • RP_Resources - Page to display three sections of NPSP information.

Visualforce Components included:

  • RP_GettingStarted - Component to include NPSP links information.
  • RP_GitHub - Component included to display Cumulus repo information from Github.
  • RP_Youtube - Component included to display videos of salesforce.org channel.

Apex classes included:

  • RP_Constants - Class to set constants values.
  • RP_GettingStartedController - Class to set values of getting started component section.
  • RP_GettingStartedTest - Test load correct values in each link.
  • RP_GitHubClient- Client to request Github information using Salesforce.org Api - From Cumulus repo.
  • RP_GitHubController - Component that brings the latest release notes from Github using RP_GitHubClient class.
  • RP_GitHubTest - Cover functionality of GitHub section.
  • RP_GitHubTestMock - Test mock that simulate a response from salesforce.org github client used in test classes.
  • RP_HTTPClient - Base HTTP client.
  • RP_YoutubeClient - Client to request Youtube videos using Salesforce.org Api - From SF.org youtube channel.
  • RP_YoutubeController- Component that retrieves youtube videos using RP_YoutubeClient class.
  • RP_YoutubeTest - Cover functionality of Youtube section.
  • RP_YoutubeTestMock - Test mock that simulate a response from salesforce.org youtube client used in test classes.
